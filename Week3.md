- Used Google's "AI Overview" to figure out how to make a nested list in Markdown.
    - Prompt: markdown nested lists
    - Answer: To create nested lists in Markdown, indent each item of the sublist with four spaces or one tab.
- Used Google's "AI Overview" to figure out how to add an image in Markdown.
    - Prompt: markdown image
    - Answer: The basic syntax for inserting an image is: \!\[Alt text for screen readers\]\(image-path-or-URL\)
- Used Google's "AI Overview" to figure out how to add a link to a directory in Markdown
    - Prompt: markdown link to directory
    - Answer: To create a markdown link to a directory, you use the standard link syntax: \[Link Text\]\(directory-path\).
